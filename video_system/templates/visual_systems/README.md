# Visual Systems

This directory contains reusable visual systems for ParlayVU video productions.

## Structure

Each subfolder represents a complete, named visual language that multiple clients can adopt (with light customization).

Current systems:
- `parlayvu_interview/` — The standard interview/podcast style (originally developed for RamAir).

## Adding New Clients

Most new clients should start here by using an existing visual system + a folder in `../client_overrides/`.

See `../docs/VISUAL_SYSTEMS_AND_CLIENT_CUSTOMIZATION.md` for the full model.

## Adding a New Visual System

Only create a new folder here when a client genuinely needs a different aesthetic that cannot be achieved through light overrides.
