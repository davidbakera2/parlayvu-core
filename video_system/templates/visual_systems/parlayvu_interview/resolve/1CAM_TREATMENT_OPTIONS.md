# 1cam Lower Third Treatment — Options

## Background

The original v1 template has a distinctive "1cam" lower third treatment:
- A smaller framed lower third area
- Explicit white border drawn around it
- Creates a "floating card" effect when in single-camera mode

## Current Direction

The user is open to **simplifying** this treatment in favor of a cleaner, smoother system.

## Proposed Options to Prototype

### Option A — Simplified (Recommended starting point)
- Full-width lower third bar (same as 2cam/3cam layouts)
- No special white border
- Consistent treatment across all layouts
- Much simpler to build and maintain in Fusion
- Cleaner, more modern feel

### Option B — Refined Border Version
- Keep a version of the bordered treatment
- Make the border more elegant / less literal than the old version
- Still use the smaller framed area
- More work in Fusion, but preserves some of the original character

### Option C — Hybrid / Switchable
- Build the lower third Fusion composition with a toggle:
  - "Standard" mode (full bar, no border)
  - "1cam Framed" mode (smaller area + border)
- This gives maximum flexibility during editing

## Recommendation

Build **Option C (Hybrid)** if it's not too much extra work in Fusion. This way you can see both versions side-by-side on real footage and decide what you prefer.

If the hybrid adds too much complexity, default to **Option A (Simplified)** and only build the bordered version later if you miss it.

## Next Step

When we start building the lower third Fusion composition, we will create both versions (or the switchable version) so you can compare them directly in a timeline.
