## Purpose

This contains the DaVinci Resolve project template for the ParlayVU interview visual system (originally developed for RamAir).

We are building this **natively inside Resolve** (Text+, Fusion, timeline architecture, Deliver presets).

- `render_presets/` — Exported render settings from the Deliver page

## Design Goals

- High visual fidelity to the existing lower thirds, b-roll cards, name cards, and 1cam treatment
- Easy to duplicate on any machine
- Scriptable by the future timeline builder
- Comfortable for human editors to work in directly
