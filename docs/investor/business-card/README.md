# ParlayVU Business Card Assets

Generated card faces are standard US business card size: 3.5 x 2.0 inches, no bleed included. PNG previews are 1050 x 600px at 300 DPI.

## Files

- `parlayvu-business-card-front.svg`: editable front SVG for Nathan Ellis, Lead Orchestrator.
- `parlayvu-business-card-back.svg`: editable back SVG for David Baker, Founder.
- `parlayvu-business-card-front-3.5x2in-300dpi.png`: high-resolution front preview.
- `parlayvu-business-card-back-3.5x2in-300dpi.png`: high-resolution back preview.
- `parlayvu-business-card-print-preview.pdf`: two-page PDF preview, front then back.

## Print Note

These files are built to final trim size only. If a printer requires bleed, extend the background/glow artwork to 3.75 x 2.25 inches and keep text/logo/photo inside a safe margin.
